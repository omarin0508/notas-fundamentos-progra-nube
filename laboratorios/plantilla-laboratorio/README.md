# Plantilla de laboratorio

Usar esta carpeta como base para crear nuevos laboratorios del curso.

## Estructura

```text
lab-XX-nombre-corto/
+-- README.md
+-- informe.md
+-- informe.pdf
+-- evidencias/
+-- recursos/
```

## Uso recomendado

1. Copiar esta carpeta y renombrarla con el formato `lab-XX-tema-corto`.
2. Completar `informe.md`.
3. Guardar capturas y resultados en `evidencias/`.
4. Guardar archivos auxiliares en `recursos/`.
5. Exportar `informe.md` a `informe.pdf` desde VS Code.

## Checklist antes de entregar

- [ ] Objetivos completos.
- [ ] Procedimiento documentado.
- [ ] Evidencias enlazadas con rutas relativas.
- [ ] Problemas y soluciones registrados.
- [ ] Conclusiones agregadas.
- [ ] PDF generado y revisado.
