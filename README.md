## Flujo recomendado

- Escribir las notas en Markdown dentro de `notas/`.
- Exportar documentos finales a `pdf/`.
- Guardar capturas y evidencias en `capturas/` o dentro del laboratorio correspondiente.
- Documentar cada laboratorio con su propio `README.md`.
- Mantener el repositorio sincronizado con GitHub.
